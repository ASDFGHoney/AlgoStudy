# 1.알고리즘과 문제 분석

- 컴퓨터 알고리즘이란??
    - step-by-step 명령어들의 집합으로 유한시간내에 컴퓨터를 이용해 문제를 해결하는 것.

- 문제 해결 순서
    1. 문제를 정의
        1. input과 output을 정한다.
    2. 문제에 대한 전략 세우기
    3. 전략의 알고리즘 서술
        1. Input
        2. Output
        3. input→output
    4. 분석
        1. correctness 정확하게 작동 하는가?
        2. Time & Space : 시간과 공간에 대한 효율성 
        3. Oprimality : 최적성 
            1. problem complexity와 algorithm complexity가 같으면 Optimal Algorithm이라고 한다.
    5. 구현
    6. 검증
